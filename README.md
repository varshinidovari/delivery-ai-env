
# AI Smart Delivery Optimization Environment

## Description
This project simulates a smart delivery system where an AI agent optimizes delivery operations based on fuel, traffic, and time.

## Features
- Delivery optimization
- Traffic simulation
- Fuel tracking
- Score tracking
- Easy, Medium, Hard difficulty

## Actions
- Deliver
- Move
- Refuel

## State
- Location
- Orders
- Fuel
- Time
- Traffic
- Score

## Goal
Maximize delivery efficiency and score.
